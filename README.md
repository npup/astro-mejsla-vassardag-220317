## Testa

### Installation

    git clone git@github.com:npup/astro-mejsla-vassardag-220317.git
    cd astro-mejsla-vassardag-220317
    npm i

### Körning

Development server, beskåda på http://localhost:3000/

    npm start

Fullskaligt bygge, byggs till `./dist`:

    npm run build
